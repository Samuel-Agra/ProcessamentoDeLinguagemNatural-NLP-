# Estudos em Processamento de Linguagem Natural (NLP)

Este repositório reúne códigos e experimentos relacionados a **Processamento de Linguagem Natural (PLN / NLP)**, com foco em fundamentos teóricos e implementações práticas.

## Áreas e Conceitos Estudados
- Teoria da Informação
- Tokenização e representação de símbolos
- Entropia e previsibilidade
- Codificação de Huffman
- Codificação aritmética
- Compressão sem perdas
- Análise estatística de texto

## Objetivo
Explorar, de forma prática, como conceitos matemáticos e estatísticos são aplicados
em NLP, compressão de dados e modelagem de informação.

> Repositório de estudo e aprendizado contínuo.
